About

Our project, undertaken for the end-year project, revolves around text detection and recognition. It encompasses several key components:

  • Utilization of a YOLOv8 model trained using the COCO-Text dataset for accurate text detection.
  • Implementation of both ResNet and VGG models, trained specifically for text recognition tasks.
  • Development of a Flask application to integrate and showcase the functionalities of our text detection and recognition system.

Our project initially faced challenges in text detection using the ICDAR2015 and SVT datasets with YOLOv8.1. Disappointed with the results, we turned to the COCO-Text dataset and YOLOv8.2. Retraining our model with a subset of 2000 images from COCO-Text improved detection accuracy. We then implemented a cropping mechanism to isolate text regions for recognition using ResNet and VGG architectures. Finally, we developed a user-friendly interface to display cropped images and recognized words, offering a complete solution for real-world text detection and recognition.
